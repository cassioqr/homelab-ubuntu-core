# Overview do Sistema

Este homelab simula um pequeno ambiente Linux em produção com conceitos de DevOps/SRE;

Isso inclui:

- Acesso seguro de SSH com MFA
- Interface de Gerenciamento Centralizada (Cockpit)
- Configuração de IP Estatico
- Design de sistema baseado em serviços
- Documentação voltada para recuperação do sistema

O sistema é mantido intencionalmente minimalista para priorizar manutenibilidade e reprodutibilidade.
